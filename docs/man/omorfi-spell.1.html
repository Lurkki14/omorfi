NAME

omorfi-spell − Spell-check Finnish words using omorfi

SYNOPSIS

omorfi-spell.sh [OPTION] [FILENAME...]

DESCRIPTION

Spell checks and corrects list of words.
−h, −−help

Print this help dialog

−V, −−version

Print version info

−v, −−verbose

Print verbosely while processing

If no FILENAMEs are given, input is read from standard input. This tool uses hfst-ospell.
